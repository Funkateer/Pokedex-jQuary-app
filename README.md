# Pokedex jQuery App
Clone of the [Pokedex-simple-js-app](https://github.com/Funkateer/Pokedex-simple-js-app) using jQuery and Bootstrap4 instead of vanilla javascript


## Screenshots

- Search for Pokemons
- Display in modals info about Pokemons
